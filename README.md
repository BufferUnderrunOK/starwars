# Hi Valstro gang!

Prerequisites: 
* NodeJS v16+
* npm / yarn

Execution Instructions

1. Start the backend server as per the original instructions. 
2. `npm install` or `yarn install`
3. `npx nest build`
4. Either:
  * `node dist/main.js` for interactive mode
  * `node dist/main.js [queryText]` to search on queryText
